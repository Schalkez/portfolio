A _sophisticated_ web application designed to **track** and _visualize_ UI changes between deployments. This tool **automatically captures screenshots** of web applications before and after each deployment, enabling _developers_ and **stakeholders** to easily identify **visual regressions** and _UI improvements_.

### **Technologies Used**

- **Next.js**: Powerful frontend framework for a _seamless experience_.
- **Firebase**: _Real-time_ data storage and _secure authentication_.
- **Google Cloud Platform**: _Scalable_ and reliable infrastructure.
- **TanStack Query**: Efficient data _fetching and caching_.
- **Jotai**: _Lightweight_ state management for _complex UI interactions_.

### **Key Features**

- **Automated Screenshots**: Captures UI snapshots _daily_ or post-deployment.
- **Visual Diff Comparison**: Identifies _UI changes_ with _intuitive visuals_.
- **Deployment History Tracking**: Stores and reviews _UI version history_.
- **Real-Time Notifications**: Instant alerts for _interface changes_.
- **Secure Authentication**: Supports _Basic Auth_ for protected access.
- **CI/CD Integration**: Triggered via _webhooks_ for automated workflows.

**Perfect** for development teams aiming to _maintain visual consistency_ and quickly detect _unintended UI changes_. This tool _streamlines development processes_ and enhances _interface quality_.
